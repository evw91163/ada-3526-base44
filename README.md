# ADA Combined Website

This repository combines all functionality from:
- evw91163/adaweb4
- evw91163/american-donut-association
- evw91163/ada

## Features
- Community Forum with categories and threads
- Events calendar
- For Sale / Wanted listings
- Links directory
- Articles
- Advertisements
- Discounts & Deals
- Help Wanted / Jobs
- FAQ section
- Admin panel
- Full-text search

Built with React + Base44 platform.
