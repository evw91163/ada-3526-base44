# American Donut Association — Combined Website

This repository combines the functionality from three source repositories:
- [adaweb4](https://github.com/evw91163/adaweb4)
- [american-donut-association](https://github.com/evw91163/american-donut-association)
- [ada](https://github.com/evw91163/ada)

## Features
- Community Forum with categories, threads, and replies
- Events calendar (upcoming & past events)
- Articles section with full article detail pages
- For Sale / Marketplace listings
- Help Wanted / Job board
- Links directory
- Advertisements section
- Discounts & Deals with promo code copy
- FAQs with search and category filtering
- Admin panel for managing content
- Responsive design with pink ADA brand theme

## Tech Stack
Built on **Base44** platform (React + Tailwind CSS + Base44 BaaS)
