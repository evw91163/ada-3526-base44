# Deployment

This app is deployed on Base44 platform.

## Live Preview
The app is available as a Base44 hosted web application.

## Architecture
- **Frontend**: React + Tailwind CSS
- **Backend**: Base44 BaaS (entities, auth, functions)
- **Database**: Base44 managed database
- **Auth**: Base44 authentication system

## Entities (Database Schema)
- ForumCategory
- ForumThread  
- ForumPost
- Event
- Article
- Listing (For Sale)
- Link
- Advertisement
- Discount
- Job (Help Wanted)
- Faq
