# ADA Combined Website - ada-3526-base44

This repository combines all functionality from:
- evw91163/adaweb4 (adaweb4)
- evw91163/american-donut-association
- evw91163/ada

## All Features Implemented

### Community
- **Forum** - Categories, threads, replies, pinning, locking
- **Events** - Submit, approve, view upcoming/past events
- **Articles** - Published articles with images

### Marketplace
- **For Sale** - Buy/sell/free listings with contact info
- **Links** - Curated resource directory with categories
- **Advertisements** - Business ads with images and links
- **Discounts** - Promo codes with copy-to-clipboard
- **Help Wanted** - Job postings with full/part-time/contract types

### User Features
- **Authentication** - Login/logout/signup flow
- **Profile** - View your posts and account info
- **Search** - Full-text search across all content types
- **FAQs** - Searchable FAQ with categories

### Admin Panel
- Approve/reject: Events, Advertisements, Discounts, Links
- Manage forum categories
- Manage FAQs

## Tech Stack
- React + Tailwind CSS
- Base44 platform (database, auth, hosting)
- shadcn/ui components
